# SnapClass – AI-Based Smart Attendance System

SnapClass is an AI-powered attendance management system that uses facial recognition technology to automate attendance tracking and eliminate proxy attendance. The platform provides dedicated teacher and student portals with secure attendance verification and real-time data management.

---

## Features

- AI-based facial recognition attendance
- Dedicated Teacher Dashboard
- Dedicated Student Dashboard
- Real-time attendance tracking
- Secure attendance verification
- Automated attendance records
- Subject-wise attendance management
- Cloud database integration using Supabase
- User-friendly interface built with Streamlit

---

## Tech Stack

- Python
- Streamlit
- Supabase
- Dlib
- Scikit-learn
- face_recognition_models

---

## Project Architecture

1. Face Detection & Recognition
2. Attendance Verification
3. Supabase Database Storage
4. Teacher Dashboard
5. Student Dashboard
6. Attendance Analytics

## Installation

### Clone Repository

```bash
git clone https://github.com/tanisha162/SNAPCLASS.git
cd SNAPCLASS
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
